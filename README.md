# Snake

Простая версия игры "Змейка" на чистом JavaScript, HTML и CSS.

Как запустить:

1. Откройте `index.html` в браузере (двойной клик или через локальный сервер).

Файловая структура:

- `index.html` — основной HTML
- `css/` — стили
- `js/` — скрипты (utils, food, snake, game)

Автор: перенесено из проекта `htmlsnake`
